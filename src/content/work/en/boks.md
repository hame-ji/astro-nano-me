### Context

- **New stack**: Discovering Angular once again in the project context, but this time also for developing a cross-platform mobile application. On the backend and backoffice side, migration to hapi and NoSQL (PostgreSQL) in a monorepo.
- **Broader technical scope**: Maintenance and evolution of the API, backoffice, and mobile application. Responsible for end-to-end modifications: from database to user interface.
- **Workflow**: Systematic integration and E2E tests on critical flows as starting points for new developments. PRs, reviews, debugging, and QA on staging environments.
- **Team**: 2 full-stack developers, 1 UX/UI designer, 1 product owner, 1 CTO.

### Examples of tasks performed

- **In-app journey optimization**: Modification of the workflow for order preparers to enable assembly of new hardware component references (QR code scans, barcodes, and testing of these components).
- **Redesign and data model reconstruction**: Overhaul of database models, [backfilling](https://atlan.com/backfilling-data-guide/) and data updates, followed by creation of summary interfaces for IoT object openings in-app, reducing customer service workload.
- **Technical documentation**: Writing documentation on Notion to explain certain technical flows for the Product Owner.

### Stack

Typescript, Angular 17/18, Capacitor, Ionic, Node, Hapi, Psql, Mocha, Cypress, Docker, Heroku, Ngrok
